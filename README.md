# Delivery App
  Online store on Python
### 1. Installation

```bash
$ git clone https://github.com/oleg-soshko/Delivery_app.git
$ cd Delivery_app
$ pip install -r requirements.txt
```

### 2. Running The App

```bash
python main.py
```

### 3. Viewing The App

Go to `http://127.0.0.1:5000`

### 4. Technology Stacks
- Python
- flask
- sqlalchemy
- HTML
- CSS
- bootstrap
